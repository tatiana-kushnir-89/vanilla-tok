# vanilla-tok

Лента коротких видео с вертикальным скроллом.

## Запуск

```bash
node server.js
```
Открываем http://localhost:3000

Нужен Node.js 18+, `npm install` не требуется.

## Структура

```
vanilla-tok/
├── videos/          # сюда кладём видео
├── js/
│   ├── app.js       # точка входа
│   ├── VideoFeed.js # лента: observer, навигация, виртуализация
│   └── VideoItem.js # один слайд: плеер, мут, прогресс
├── css/style.css
├── index.html
└── server.js
```

## Про реализацию

Скролл сделан через CSS `scroll-snap` — браузер сам занимается анимацией и жестами, никакого JS для этого не нужно.

За воспроизведением следит `IntersectionObserver`: когда слайд занимает больше 75% экрана — видео играет, остальные стоят на паузе.

Чтобы не грузить всё сразу, видео по умолчанию имеют `preload="none"`. Соседние слайды подгружаются заранее через `requestIdleCallback` — в момент, когда браузер всё равно простаивает. Слайды, которые далеко от текущего, выгружаются из памяти полностью.

Прогресс-бар обновляется через `requestAnimationFrame` — плавно и без лишней нагрузки.

Сервер поддерживает `Range`-запросы, так что видео стримится кусками, а не скачивается целиком перед воспроизведением.

## Управление

- **Свайп / скролл** — следующее и предыдущее видео
- **Клик по видео** — play / pause
- **Кнопка звука** — мут / анмут
- **Стрелки справа** — навигация (только на десктопе)
- **Клавиши ↑↓ или j/k** — навигация с клавиатуры
- **M** — глобальный мут
